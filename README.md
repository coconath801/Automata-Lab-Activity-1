# Automata-Lab-Activity-1
DFA String Acceptance Checker
<img width="1366" height="768" alt="image" src="https://github.com/user-attachments/assets/3708fab0-cc34-4db0-8c5e-67277cd54479" />
A Java program that simulates a Deterministic Finite Automaton (DFA) for binary strings ending with “01”.

Continuously accepts user input and checks each string until the user types “exit”.

Displays “Accepted” if the string ends with 01, otherwise “Rejected”.

Demonstrates basic automata simulation using Java control structures and loops.
